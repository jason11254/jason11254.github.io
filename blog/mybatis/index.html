SpringBoot 整合 MyBatis 環境設定

By 巫宗霖 2022-03-02

MyBatis是一個優秀的持久層框架，它對jdbc的操作資料庫的過程進行封裝，使開發者只需要關注 SQL 本身，而不需要花費精力去處理例如註冊驅動、創建connection、創建statement、手動設置參數、結果集檢索等jdbc繁雜的過程代碼。

相比Hibernate完全ORM的框架，MyBatis在查詢上因為是工程師手動寫SQL，所以靈活性較高。

Spring Boot 中的 MyBatis設定

  1. pom.xml 中加入導入包。

    mysql：

    <dependency>
    <groupId>mysql</groupId>
    <artifactId>mysql-connector-java</artifactId>
    <scope>runtime</scope>
</dependency>

    mybatis：

    <dependency>
    <groupId>org.mybatis.spring.boot</groupId>
    <artifactId>mybatis-spring-boot-starter</artifactId>
    <version>2.2.1</version>
</dependency>

    Spring boot的一些導入包(本體starter與test建立專案時已經建立)：

    <dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
</dependency>
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-jdbc</artifactId>
</dependency>

    其他：lombok

    <dependency>
    <groupId>org.projectlombok</groupId>
    <artifactId>lombok</artifactId>
    <version>1.18.22</version>
    <scope>provided</scope>
</dependency>

  2. 配置文件 application.properties

    # mysql setting
pring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
spring.datasource.url=jdbc:mysql://localhost:3306/[DB名稱]?serverTimezone=Asia/Taipei&characterEncoding=utf-8
spring.datasource.username=[帳號]
spring.datasource.password=[密碼]

# mybatis setting
mybatis.type-aliases-package=com.example.mybatistest.model # 這一行設定物件class位置，這樣寫Mapping文件時，就不用打出完整路徑了。
mybatis.mapper-locations=classpath:mybatis/mapper/*.xml # 這一行設定mapper.xml的位置，這樣mybatis才找的到sql在哪裡。

  3. 比照上面設定檔在 com/example/mybatistest 路徑建立資料夾 model，並創建對應 DB Table 的物件class。(請先去DB建立好Table再來撰寫)

    範例：Student.java

    package com.example.mybatistest.model;

import lombok.AllArgsConstructor;
import lombok.Data;
import lombok.NoArgsConstructor;

@Data
@NoArgsConstructor
@AllArgsConstructor
public class Student {
    private Integer id;
    private String name;

}

  4. 在 com/example/mybatistest 路徑建立資料夾 mapper，並創建介面Mapper。且須在上方加上註解@Mapper與@Repository(@Component)。

    然後便可以開始寫一些需要mybatis執行的方法。

    範例：StudentMapper.java

    package com.example.mybatistest.mapper;

import com.example.mybatistest.model.Student;
import org.apache.ibatis.annotations.Mapper;
import org.springframework.stereotype.Component;
import org.springframework.stereotype.Repository;

import java.util.List;

@Mapper // 這個註解表示這是一個 mybatis mapper
@Repository
//@Component 這個與 @Repository 二選一
public interface StudentMapper{
    List<Student> queryStudentList();
    Student getById(Integer id);
}

  5. 比照第2點的設定檔，在 resources 資料夾下建立資料夾mybatis，後在mybatis下再建立資料夾mapper。

    於mapper下建立Mapper.xml檔案(檔名通常會與介面名稱一樣)，並在裡面編寫SQL。

    範例：StudentMapper.xml

    <?xml version="1.0" encoding="UTF-8" ?>
<!DOCTYPE mapper
        PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-mapper.dtd">
<!-- 這裡要填入mapper.java的位置 -->
<mapper namespace="com.example.mybatistest.mapper.StudentMapper">

    <!-- 
    	如果 application.properties 沒設定 mybatis.type-aliases-package 的話，
    	這裡的 resultType 會變成 com.example.mybatistest.model.Student，
    	而不是只寫Student。
    -->
    <select id="queryStudentList" resultType="Student"> 
        SELECT * FROM student
    </select>

    <select id="getById" parameterType="int" resultType="Student">
        SELECT * FROM student WHERE id=#{id}
    </select>

</mapper>

  6. 然後開始測試，這裡找其中一個方法來測。

    建立Service：

    package com.example.mybatistest.service;

import com.example.mybatistest.model.Student;

public interface StudentService {
    public Student getById(Integer id);
}

    package com.example.mybatistest.service;

import com.example.mybatistest.mapper.StudentMapper;
import com.example.mybatistest.model.Student;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.stereotype.Component;

@Component
public class StudentServiceImpl implements StudentService{

    @Autowired
    StudentMapper studentMapper;

    @Override
    public Student getById(Integer id) {
        Student student = studentMapper.getById(id);
        return student;
    }
}

    建立Controller：

    package com.example.mybatistest.controller;

import com.example.mybatistest.model.Student;
import com.example.mybatistest.service.StudentService;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.PathVariable;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class StudentController {

    @Autowired
    private StudentService studentService;

    @RequestMapping("/students/{id}")
    public Student getById(@PathVariable Integer id){
        Student student = studentService.getById(id);
        return student;
    }

}

    執行後於瀏覽器輸入：http://localhost:8080/students/2 即可得到回傳的結果。
